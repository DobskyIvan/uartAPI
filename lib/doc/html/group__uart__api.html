◆ uartInit()

eErrorCode uartInit ( sUartApi * uartApi,
eErrorCode(* init )(void) )

Функция инициализации объекта API.

Note
Эта функция выделяет память. Для завершения работы интерфейса необходимо вызывать uartDeinit(), для ее освобождения
Пример вызова:
sUartApi uart1;
eErrorCode bar (void){
//do some platform-specific stuff
return E_NOERR;
}
uartInit(&uart1, &bar);
// do something usefull
uartDeinit(&uart1);
